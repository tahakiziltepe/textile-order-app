# Textile Order App

Bu proje, tekstil siparişlerini yönetmek için oluşturulmuş bir React uygulamasıdır. Vite, TypeScript ve Tailwind CSS kullanılarak geliştirilmiştir.

## ✨ Özellikler

- Sipariş oluşturma, görüntüleme, güncelleme ve silme.
- Tedarikçi yönetimi.
- Sipariş takibi.
- Analitik sayfası ile sipariş verilerini görselleştirme.

## 🚀 Kullanılan Teknolojiler

- [React](https://reactjs.org/) - Kullanıcı arayüzü kütüphanesi
- [Vite](https://vitejs.dev/) - Hızlı geliştirme ortamı
- [TypeScript](https://www.typescriptlang.org/) - JavaScript için statik tip denetleyicisi
- [React Router](https.com/tahakiziltepe/textile-order-app.git)

## 📦 Kurulum

Projenin çalıştırılabilmesi için öncelikle gerekli bağımlılıkların yüklenmesi gerekmektedir. Bunu yapmak için aşağıdaki adımları izleyin:

1. Proje dizinine gidin:
   ```sh
   cd textile-order-app
   ```
2. Gerekli paketleri yükleyin:
   ```sh
   npm install
   ```

### Projeyi Çalıştırma

Projeyi geliştirme modunda çalıştırmak için:
```sh
npm run dev
```
Uygulama `http://localhost:5173` adresinde açılacaktır.

## 📜 Kullanılabilir Scriptler

- `npm run dev`: Geliştirme sunucusunu başlatır.
- `npm run build`: Üretim için projeyi derler.
- `npm run preview`: Üretim derlemesini yerel olarak önizler.
- `npm run deploy`: Projeyi GitHub Pages'e dağıtır.
